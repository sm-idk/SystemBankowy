## Jak uruchomić

Uruchom program:
```bash
java Bank.java
```


## Funkcjonalności

- Bankowość internetowa (przelewy, sprawdzanie salda, historia transakcji)
- Obsługa bankomatu (wpłaty, wypłaty, zmiana PIN)
- Panel administratora (podgląd klientów i transakcji)
